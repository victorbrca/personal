   2016-10-19  

Here’s a quick function that will display a prompt with timeout in a bash script: _myCountdownFunction (){echo-e "Hit \"Ctrl+c\" to quit or \"Enter\" to continue... \c"cnt=5while((cnt >=0));doif((cnt < 9));thenecho-e "\b\b${cnt}s\c"elif((cnt==9));thenecho-e "\b\b\b ${cnt}s\c"elif((cnt <=99));thenecho-e "\b\b\b\b ${cnt}s\c"elif((cnt < 999));thenecho-e "\b\b\b\b${cnt}s\c"firead-t 1my_reply (($?==0))&&exit1letcnt-=1done} The user will see the following message on his terminal with the seconds counting down in place: john@ubuntu:~$ Hit "Ctrl+c" to quit or "Enter" to continue... 5s At the end of the specified time in ‘cnt’, the script (where the function would be) will continue executing, and if the user hits “Ctrl+c” before that, it will exit (both script and function).
